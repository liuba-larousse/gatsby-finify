## Run it locally

```bash
# Install dependencies
npm install # or yarn

# Run in dev mode
cd finify_expert
gatsby develop # or yarn dev

# Run tests
gatsby test # or yarn test
```

- **Development**: [React](https://reactjs.org/) ,[Gatsby](https://www.gatsbyjs.com/), [SCSS](https://sass-lang.com/)
